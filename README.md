📁 Portafolio Web Personal

Este proyecto es un portafolio web responsive donde presento mi perfil como desarrollador, incluyendo una galería temática de mascotas 🐾, enlaces a mis redes sociales y un formulario de contacto.

🚀 Características principales
📱 Diseño Responsive
Adaptado a dispositivos móviles, tablets y escritorio.
🐶 Galería de mascotas
Sección visual con imágenes organizadas de forma atractiva.
📌 Sidebar interactivo
Barra lateral con accesos directos a mis redes sociales.
✉️ Formulario de contacto
Permite a los usuarios enviarme mensajes fácilmente.
🧱 HTML semántico
Uso correcto de etiquetas como header, nav, main, section, article, footer para mejorar accesibilidad y SEO.
🛠️ Tecnologías utilizadas
HTML5
CSS3
JavaScript
